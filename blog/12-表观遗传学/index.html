表观遗传学

DNA 甲基化

定义：DNA 甲基化是一种发生在 DNA 序列上的化学修饰，可以在转录及细胞分裂前后稳定的遗传。DNA 甲基化是重要的表观遗传学修饰之一。在哺乳动物中，DNA 甲基化主要发生在 CpG 二核苷酸中的胞嘧啶的第五个碳原子上。DNA 甲基化对转录的调控起着十分重要的作用，如 DNA 甲基化阻碍转录因子的结合、通过排斥组蛋白甲基化转移酶的结合来阻止活性染色质标记、影响核小体的定位等。DAN 甲基化还有许多重要的意义，如 DNA 甲基化行使重复元件沉默的功能，使基因组能够长期稳定；DNA 甲基化行使染色体的选择性沉默功能，使 X 染色体随机失活和维持基因印记；DNA 甲基化行使基因的组织特异性表达功能，可以抑制生殖细胞的关键基因而迫使细胞进入分化过程。

CpG 岛

定义：在哺乳动物中，约 60～90%的 CpG 二核苷酸是被甲基化的，而未被甲基化的 CpG 二核苷酸聚集成簇形成的区域称为 CpG 岛。CpG 岛的主要特点是 GC 含量及 CpG 含量非常高且大部分处于非甲基化状态，其主要分布在基因的启动子、5’非编码区以及第一外显子区域。CpG 岛的存在，说明该区域是处于非甲基化的状态，可以避免甲基化的 CpG 带来的高突变率。

CpG 甲基化：
CpG 甲基化只要发生在 C 上的第五个碳原子上，且最多可以发生 3 个甲基化。正常情况下，由于基因组的错配修复可以精确识别并修复 C 的脱氨基产物，但甲基化的 C 的脱氨基产物不会被识别而发生缓慢的突变最终变为 T，并且在生殖细胞中可以遗传。

异常 DNA 甲基化与癌症发生

主要有三类：

  1. 基因组整体低甲基化：在癌症基因组中发生全局性的基因组去甲基化，这一现象被称为超低甲基化（hypomethylation）
  2. 印记丢失（loss of imprinting）：基因组印记需要父母本等位基因中的一条发生超甲基化而建立单等位基因表达的模式，但在癌症基因组中某些印记基因的等位特异性甲基化模式发生了异常，使得原先被印记的无活性的等位基因被重新激活。
  3. 基因超甲基化：在癌症基因组中使得抑癌基因被沉默的异常现象被称为基因超甲基化
  4. 可见在癌症基因组中，超低甲基化使得癌基因被激活转录、超甲基化使得抑癌基因被抑制转录
